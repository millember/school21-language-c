# Pong Game in C

Классическая игра Pong для терминала с ASCII-графикой. Два режима: пошаговый и интерактивный.

## Быстрый старт

### Пошаговый режим:
```bash
gcc src/pong.c -o pong
./pong
```

### Интерактивный режим (требует ncurses):
```bash
gcc src/pong_interactive.c -lncurses -o pong_interactive
./pong_interactive
```

## Управление

**Игрок 1 (лево):** A/Z (вверх/вниз)  
**Игрок 2 (право):** K/M (вверх/вниз)  
**Пропуск хода:** Space (в пошаговом режиме)

## Характеристики

- **Поле:** 80×25 символов
- **Ракетки:** 3 символа (#)
- **Мяч:** 1 символ (o)
- **Победа:** 21 очко
- **Графика:** ASCII-символы

## Структура

```
pong/
├── src/
│   ├── pong.c              # Пошаговая версия
│   └── pong_interactive.c  # Интерактивная версия (ncurses)
└── README.md
