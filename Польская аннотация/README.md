# Math Function Plotter in C

Программа для построения графика математических функций в терминале с использованием ASCII-графики. Поддерживает построение графиков тригонометрических, логарифмических и алгебраических функций.

## Быстрый старт

### Компиляция:
```bash
make
```

### Запуск:
```bash
./../build/Quest_1
```

### Очистка:
```bash
make clean
```

### Пересборка:
```bash
make rebuild
```

### Проверка стиля кода:
```bash
make clang
```

## Поддерживаемые функции

- **sin(x)** - синус
- **cos(x)** - косинус  
- **tan(x)** - тангенс
- **ctg(x)** - котангенс
- **sqrt(x)** - квадратный корень
- **ln(x)** - натуральный логарифм
- **+** - сложение
- **-** - вычитание
- **\*** - умножение
- **/** - деление
- **~** - унарный минус

## Характеристики

- **Область определения**: от 0 до 4π (примерно 12.566)
- **Область значений**: от -1 до 1 (но масштабируется автоматически)
- **Размер поля**: 80×25 символов
- **Ось X**: горизонтальная, 80 точек
- **Ось Y**: вертикальная, 25 точек
- **Графика**: ASCII-символы ('*' для точек графика, '.' для фона)

## Примеры ввода

```bash
# Простая функция
sin(x)

# Комбинированная функция
sin(x)+cos(x)

# Сложное выражение
sin(x)*cos(x)

# С использованием констант
sin(x)+0.5

# Вложенные функции
sin(cos(x))

# Логарифмическая функция
ln(x+1)
```

## Описание файлов

### main.c
Основной файл программы. Содержит точку входа и координацию работы модулей:
- Получение ввода от пользователя
- Преобразование выражения в польскую нотацию
- Расчет значений функции
- Построение графика

### polish_note.c/h
Реализация алгоритма преобразования выражения в обратную польскую нотацию (RPN) и его вычисления:
- Поддержка унарного минуса
- Обработка математических функций
- Приоритеты операций
- Расчет значений с учетом переменной x

### input.c/h
Модуль обработки пользовательского ввода:
- Валидация введенного выражения
- Распознавание математических функций
- Проверка синтаксиса

### graph.c/h
Модуль графического вывода:
- Инициализация поля 80x25
- Построение точек графика
- Вывод результата в терминал

### lib_struct.h
Структура данных "стек" для алгоритма преобразования в польскую нотацию

## Особенности реализации

1. **Преобразование в польскую нотацию** - выражение преобразуется в обратную польскую нотацию для упрощения вычислений
2. **Валидация ввода** - проверка корректности математического выражения
3. **Обработка ошибок** - корректная обработка математических ошибок (деление на ноль, логарифм от неположительного числа и т.д.)
4. **Масштабирование** - автоматическое масштабирование графика для отображения в диапазоне [-1, 1]
5. **Структурное программирование** - разделение на модули с четкими интерфейсами

## Ограничения

1. Выражение должно вводиться **без пробелов**
2. Поддерживаются только указанные математические функции
3. Область определения фиксирована: [0, 4π]
4. Размер поля фиксирован: 80×25 символов
