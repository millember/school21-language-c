# Game of Life in C

Классическая игра "Жизнь" (Conway's Game of Life) для терминала с ASCII-графикой. Визуализация клеточного автомата с интерактивным управлением скоростью.

## Быстрый старт

### Компиляция с ncurses:
```bash
gcc src/game_of_life.c -lncurses -o game_of_life
```

### Запуск с различными начальными состояниями:
```bash
# Запуск с фигурой "Сердце"
./game_of_life < hearts.txt

# Запуск с фигурой "Мигалка" (Blinker)
./game_of_life < blinker.txt

# Запуск с фигурой "Глайдер"
./game_of_life < glider.txt

# Запуск с фигурой "Пульсар"
./game_of_life < pulsar.txt

# Запуск с фигурой "Пушка Госпера"
./game_of_life < gosper.txt

# Запуск с фигурой "Корабль"
./game_of_life < ship.txt

# Запуск с фигурой "Блок"
./game_of_life < block.txt

# Запуск с шахматным паттерном
./game_of_life < shahmat.txt
```

## Управление

- **A/a**: Увеличить скорость (уменьшить задержку на 10мс)
- **Z/z**: Уменьшить скорость (увеличить задержку на 10мс)
- **Пробел**: Выйти из игры

## Характеристики

- **Поле**: 80×25 клеток
- **Закрытые границы**: торическое поле (зациклено по краям)
- **Клетки**: '#' (живая) и '.' (мертвая)
- **Скорость**: регулируется от 10мс до 1000мс
- **Графика**: ASCII-символы через библиотеку ncurses

## Правила игры (Conway's Game of Life)

1. Любая живая клетка с менее чем двумя живыми соседями умирает (одиночество)
2. Любая живая клетка с двумя или тремя живыми соседями выживает
3. Любая живая клетка с более чем тремя живыми соседями умирает (перенаселение)
4. Любая мертвая клетка с тремя живыми соседями становится живой (размножение)
